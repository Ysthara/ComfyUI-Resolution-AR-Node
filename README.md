# ComfyUI Resolution & AR Calculator

A simple ComfyUI custom node that calculates width and height based on:

- Long side (1K or 2K)
- Aspect ratio (1:1, 2:3, 16:9, etc.)
- Orientation (auto / landscape / portrait)

## Installation (with ComfyUI-Manager)

1. Open ComfyUI-Manager in your ComfyUI.
2. Go to "Install via URL".
3. Paste this repo URL.
4. Restart ComfyUI.
